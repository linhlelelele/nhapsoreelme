📖 Hướng dẫn lấy Meta Access Token

1
Tạo Meta App tại developers.facebook.com

My Apps → Create App → Use case: Other → App type: Business → đặt tên (vd: AdsLens Personal)

2
Add Marketing API vào App

Trong dashboard App → tìm "Marketing API" → click Set up

3
Lấy Access Token tại Graph API Explorer

Chọn Meta App của bạn → Generate Access Token → cấp permissions: ads_read, business_management → copy token (bắt đầu bằng EAA...)

4
Đổi sang Long-Lived Token (sống 60 ngày)

Tại Graph API Explorer, click ℹ️ bên cạnh token → Open in Access Token Tool → Extend Access Token → copy token mới

5
Lấy Ad Account ID

Tại Business Manager → Settings → Ad Accounts → click TK → ID có dạng act_1234567890

6
Paste vào AdsLens

Token + 2 Ad Account ID → click "Lưu config" để browser nhớ → click "Pull data" để load

⚠ Bảo mật: Token được lưu trong localStorage của browser, KHÔNG gửi đi đâu khác ngoài Meta API. Token chỉ visible trên máy bạn. Nếu dùng máy chung, nhớ xóa config sau khi xong.
